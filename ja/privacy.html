プライバシーポリシー

最終更新: 2026年5月

Vwake（以下「本アプリケーション」）は、Hocatech が開発・運営しています。当社はお客様のプライバシーを尊重します。本ポリシーは、当社がお客様の情報をどのように収集・利用・保護するかについて説明します。

1. データの収集と保存

1.1 ローカルデータ: アラーム設定、ニックネーム、各種設定は、お客様の端末内のローカル SQLite データベースに保存されます。

1.2 クラウドデータ: 購入情報の同期、アプリ内仮想通貨（V-Coin）の機能、および端末間でのアカウント復元を実現するため、本アプリケーションは Firebase Authentication と Google Cloud Firestore を使用し、以下の情報を保存します。

匿名アカウント（初回起動時にデフォルトで自動作成）：

  • Firebase 匿名ユーザー ID
  • V-Coin 残高
  • 購入済みボイスパックの記録
  • 紹介コードの利用記録（使用された場合）

連携アカウント（「設定 → アカウントを連携」から Google または Apple を連携した場合、上記に加えて以下を保存）：

  • メールアドレス（Apple の「メールを非公開」機能を利用された場合は、Apple が提供する中継アドレスのみを保存）
  • 表示名（Apple 連携の初回のみ取得され、以降のサインインでは提供されません）
  • 連携済みサインインプロバイダーの一覧（例: google.com、apple.com）
  • アカウント連携日時

アカウント連携は任意の機能です。端末の機種変更や再インストール後も、同じ Google / Apple アカウントで V-Coin やボイスパックを復元できるようにするためのものです。アプリ内からいつでもサインアウトやアカウント削除を行えます。

1.3 プッシュトークン（FCM Token）: アプリケーションが OS によって一時停止された場合でもアラームを時刻どおりに鳴らすため、本アプリケーションは端末の Firebase Cloud Messaging プッシュトークンを Firestore の pending_alarms コレクションに保存します。このトークンは本アプリケーションのアラーム通知のみに使用され、他のサービスや広告プラットフォームと共有されることはありません。

1.4 端末情報: 本アプリケーションは、デバッグおよび互換性確認のため、基本的な端末情報（端末モデル、OS バージョンなど）を収集する場合があります。

2. 第三者サービス

本アプリケーションは、以下の Google Firebase サービスを利用しています。

  • Firebase Authentication（匿名認証、Sign in with Google、Sign in with Apple）
  • Cloud Firestore（クラウドデータ同期）
  • Cloud Functions（サーバー側での購入検証、精算、アカウント削除などの処理）
  • Firebase Cloud Messaging（アラーム通知）
  • Firebase Storage（ボイスパック音声ファイルのダウンロード）

上記 Firebase サービスは Google プライバシーポリシー（https://policies.google.com/privacy）に準拠します。

「Google と連携」機能をご利用の場合、お客様と Google との間のデータのやり取りには、別途 Google プライバシーポリシーが適用されます。本アプリケーションが Google に対して要求する情報は、お客様のメールアドレスと基本的なプロフィール情報（表示名など）のみです。

「Apple と連携」機能をご利用の場合、お客様と Apple との間のデータのやり取りには、別途 Apple プライバシーポリシー（https://www.apple.com/legal/privacy/）が適用されます。Apple の「メールを非公開」機能をご利用の場合、本アプリケーションが取得するのは Apple が生成した中継メールアドレスのみであり、お客様の実際のメールアドレスを知ることはできません。

本アプリケーションは、広告 SDK や第三者によるトラッキング・解析ツールを一切使用していません。

3. 購入情報

アプリ内課金は、Apple App Store または Google Play Store によって処理されます。当社はお客様の決済情報（クレジットカード番号など）にアクセスしたり保存したりすることはありません。購入記録は、複数端末間での同期のため、ボイスパック ID のみが Firestore に保存されます。

4. 音声ファイル

無料ボイスパックの音声ファイルは、本アプリケーションに同梱されています。有料ボイスパックの音声ファイルは、購入後に Firebase Storage 経由でお客様の端末にダウンロードされます。ダウンロード後、音声ファイルはローカルに保存され、再アップロードされることはありません。

5. データセキュリティ

当社は、Firebase が提供するセキュリティ機構（通信経路の暗号化（TLS）および Firestore セキュリティルールを含む）を用いて、クラウドデータを保護します。ローカルデータは、お客様の端末の OS が提供するセキュリティ機構によって保護されます。

6. データの保持と削除

6.1 クラウドデータは、お客様が削除を申請されるまで保持されます。

6.2 アプリ内からいつでもアカウントを削除できます（経路: 設定ページ → アカウントを削除）。確認後、サーバーは直ちに以下を実行します。

  • users/{uid} ドキュメントのすべてのデータ（V-Coin 残高、購入済みボイスパックの記録、連携サインイン情報など）を削除
  • pending_alarms コレクション内のお客様の未配信アラーム記録をすべて削除
  • Firebase Authentication 内のお客様のユーザーレコードを削除

完了後は、新しい匿名ユーザーとして本アプリケーションを引き続きご利用いただけます。削除されたアカウントのデータを復元することはできません。

6.3 アプリにアクセスできない場合は、hello@vwake.app より削除をご申請いただくこともできます。当社は 30 日以内に削除を完了します。

6.4 本アプリケーションをアンインストールすると、すべてのローカルデータは削除されますが、クラウドデータは上記いずれかの方法で削除する必要があります。

7. 子どものプライバシー

本アプリケーションは、13 歳未満の子どもを対象として設計されておらず、当社は子どもの個人情報を意図的に収集することはありません。保護者の同意なく子どもが本アプリケーションを使用していることに気付かれた場合は、当社までご連絡ください。速やかに関連データを削除いたします。

8. 本ポリシーの変更

本プライバシーポリシーに重要な変更が生じた場合、アプリケーションのアップデートを通じてお知らせいたします。変更後も本アプリケーションを継続してご利用いただくことは、更新されたポリシーへの同意とみなされます。

9. お問い合わせ

プライバシーに関するご質問は、hello@vwake.app までご連絡ください。
