VwakePrivacy Policy

隱私權政策

最後更新日期：2026 年 4 月
← 返回 Hocatech 首頁

Vwake（以下稱「本應用程式」）由 Hoca Tech 開發與營運。我們尊重您的隱私，本政策說明我們如何收集、使用和保護您的資訊。

1. 資料收集與儲存

1.1 本機資料：鬧鐘設定、暱稱和偏好設定儲存在您的裝置本機 SQLite 資料庫中。

1.2 雲端資料：為提供購買同步與虛擬貨幣（V 幣）功能，本應用程式使用 Firebase Authentication 建立匿名帳號，並透過 Google Cloud Firestore 儲存以下資訊：

  • Firebase 匿名使用者 ID
  • V 幣餘額
  • 已購買的語音包記錄

1.3 裝置資訊：本應用程式可能收集基本裝置資訊（如裝置型號、作業系統版本）以用於除錯與相容性用途。

2. 第三方服務

本應用程式使用以下 Google Firebase 服務：

  • Firebase Authentication（匿名帳號驗證）
  • Cloud Firestore（雲端資料同步）
  • Firebase Storage（語音包音訊檔案下載）

上述服務受 Google 隱私權政策約束（https://policies.google.com/privacy）。

本應用程式不使用任何廣告 SDK 或第三方追蹤/分析工具。

3. 購買資訊

應用程式內購買由 Apple App Store 或 Google Play Store 處理。我們不會存取或儲存您的付款資訊（如信用卡號碼）。購買記錄僅以語音包 ID 形式儲存於 Firestore，用於跨裝置同步。

4. 音訊檔案

免費語音包音訊檔案包含在應用程式內。付費語音包音訊檔案於購買後透過 Firebase Storage 下載至您的裝置。下載完成後，音訊檔案儲存於本機，不會再次上傳。

5. 資料安全

我們採用 Firebase 提供的安全機制保護您的雲端資料，包括傳輸加密（TLS）及 Firestore 安全規則。本機資料受您裝置作業系統的安全機制保護。

6. 資料保留與刪除

6.1 雲端資料將持續保留至您主動要求刪除為止。

6.2 如需刪除帳號及所有相關雲端資料，請透過下方聯繫方式與我們聯繫，我們將在 30 日內完成刪除。

6.3 解除安裝應用程式將刪除所有本機資料，但雲端資料需另行申請刪除。

7. 兒童隱私

本應用程式不針對 13 歲以下兒童設計，亦不會有意收集兒童之個人資訊。若您發現有兒童在未經家長同意下使用本應用程式，請聯繫我們，我們將立即刪除相關資料。

8. 變更通知

如有隱私權政策重大變更，我們將透過應用程式更新通知您。繼續使用本應用程式即視為接受更新後之政策。

9. 聯繫方式

如有任何隱私權相關問題，請透過 contact@hocatech.com 聯繫我們。
