隱私權政策

最後更新日期：2026 年 5 月

Vwake（以下稱「本應用程式」）由 Hoca Tech 開發與營運。我們尊重您的隱私，本政策說明我們如何收集、使用和保護您的資訊。

1. 資料收集與儲存

1.1 本機資料：鬧鐘設定、暱稱和偏好設定儲存在您的裝置本機 SQLite 資料庫中。

1.2 雲端資料：為提供購買同步、虛擬貨幣（V 幣）功能及跨裝置帳號恢復，本應用程式使用 Firebase Authentication 與 Google Cloud Firestore 儲存以下資訊：

匿名帳號（預設，首次啟動時自動建立）：

  • Firebase 匿名使用者 ID
  • V 幣餘額
  • 已購買的語音包記錄
  • 推薦碼使用紀錄（若有使用）

連結帳號（您主動於「設定 → 連結帳號」連結 Google 或 Apple 後額外儲存）：

  • 電子郵件地址（使用 Apple「隱藏我的電子郵件」時，僅儲存 Apple 提供之中繼地址）
  • 顯示名稱（僅 Apple 首次連結時取得，後續登入不再提供）
  • 已連結之登入提供者清單（例如 google.com、apple.com）
  • 帳號連結時間戳記

連結帳號為選用功能，目的是讓您在更換裝置或重新安裝後，可透過同一個 Google / Apple 帳號恢復 V 幣與語音包。您可隨時於應用程式內登出或刪除帳號。

1.3 推播權杖（FCM Token）：為確保鬧鐘於應用程式被系統暫停時仍能準時觸發，本應用程式會儲存裝置的 Firebase Cloud Messaging 推播權杖至 Firestore 之 pending_alarms 集合。此權杖僅用於本應用程式自身的鬧鐘推播，不會分享至其他服務或廣告平台。

1.4 裝置資訊：本應用程式可能收集基本裝置資訊（如裝置型號、作業系統版本）以用於除錯與相容性用途。

2. 第三方服務

本應用程式使用以下 Google Firebase 服務：

  • Firebase Authentication（匿名驗證、Sign in with Google、Sign in with Apple）
  • Cloud Firestore（雲端資料同步）
  • Cloud Functions（伺服器端購買驗證、結算與帳號刪除等運算）
  • Firebase Cloud Messaging（鬧鐘推播）
  • Firebase Storage（語音包音訊檔案下載）

上述 Firebase 服務受 Google 隱私權政策約束（https://policies.google.com/privacy）。

當您使用「連結 Google」功能時，您與 Google 之間的資料交換另受 Google 隱私權政策約束。本應用程式僅向 Google 索取您的電子郵件地址與基本個人資料（如顯示名稱）。

當您使用「連結 Apple」功能時，您與 Apple 之間的資料交換另受 Apple 隱私權政策約束（https://www.apple.com/legal/privacy/）。您可以使用 Apple 提供之「隱藏我的電子郵件」功能，本應用程式僅會取得 Apple 產生之中繼電子郵件地址，無法得知您的真實郵件地址。

本應用程式不使用任何廣告 SDK 或第三方追蹤/分析工具。

3. 購買資訊

應用程式內購買由 Apple App Store 或 Google Play Store 處理。我們不會存取或儲存您的付款資訊（如信用卡號碼）。購買記錄僅以語音包 ID 形式儲存於 Firestore，用於跨裝置同步。

4. 音訊檔案

免費語音包音訊檔案包含在應用程式內。付費語音包音訊檔案於購買後透過 Firebase Storage 下載至您的裝置。下載完成後，音訊檔案儲存於本機，不會再次上傳。

5. 資料安全

我們採用 Firebase 提供的安全機制保護您的雲端資料，包括傳輸加密（TLS）及 Firestore 安全規則。本機資料受您裝置作業系統的安全機制保護。

6. 資料保留與刪除

6.1 雲端資料將持續保留至您主動要求刪除為止。

6.2 您可於應用程式內隨時刪除帳號（路徑：設定頁面 → 刪除帳號）。確認刪除後，伺服器將立即執行以下動作：

  • 刪除您在 users/{uid} 文件中的所有資料（V 幣餘額、已購買的語音包記錄、登入方式記錄等）
  • 刪除您在 pending_alarms 集合中所有待發鬧鐘紀錄
  • 刪除您在 Firebase Authentication 中的使用者紀錄

完成後，您將以全新的匿名身份繼續使用本應用程式，原帳號之資料無法復原。

6.3 若您無法存取應用程式，亦可透過 hello@vwake.app 申請刪除，我們將在 30 日內完成。

6.4 解除安裝應用程式將刪除所有本機資料，但雲端資料需透過上述方式刪除。

7. 兒童隱私

本應用程式不針對 13 歲以下兒童設計，亦不會有意收集兒童之個人資訊。若您發現有兒童在未經家長同意下使用本應用程式，請聯繫我們，我們將立即刪除相關資料。

8. 變更通知

如有隱私權政策重大變更，我們將透過應用程式更新通知您。繼續使用本應用程式即視為接受更新後之政策。

9. 聯繫方式

如有任何隱私權相關問題，請透過 hello@vwake.app 聯繫我們。
